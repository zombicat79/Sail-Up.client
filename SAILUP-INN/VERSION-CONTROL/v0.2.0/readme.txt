SailUp BO

HOW TO RUN THE SERVER
---------------------
1. Open terminal.app
2. Paste this to open the project folder: cd ~/Desktop/sailup-bo
3. Paste this to run the server: node server.js
4. Open the web browser and paste this URL: http://localhost:3000

HOW TO RESET THE SERVER
-----------------------
1. Open terminal.app
2. Press CTRL+C to stop the server
3. Paste this to run the server again: node server.js
4. Press CMD+SHIFT+R to refresh the application in the browser
