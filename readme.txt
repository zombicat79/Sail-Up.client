SailUp BO

HOW TO RUN THE SERVER
---------------------
1. Open terminal.app
2. Paste this to open the project folder: cd ~/Desktop/Sailup
3. Paste this to run the server: node server/app.js
4. Run SailUp: Open the web browser and paste this URL: http://localhost:3000/main
5. Run SailUp Inn: Open the web browser and paste this URL: http://localhost:3000/inn

HOW TO RESET THE SERVER
-----------------------
1. Open terminal.app
2. Press CTRL+C to stop the server
3. Paste this to run the server again: node server/app.js
4. Press CMD+SHIFT+R to refresh the application in the browser
